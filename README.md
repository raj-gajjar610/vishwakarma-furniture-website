# Vishwakarma Furniture – Web Application

**Vishwakarma Furniture** is a dynamic web application developed to simplify the process of browsing and managing furniture products online. The website provides an attractive and user-friendly interface where customers can explore furniture categories, view product details, add items to a wishlist or shopping cart, and place orders.

The application also includes an admin panel for managing categories, subcategories, products, customers, and orders efficiently. The project follows the MVC (Model-View-Template) architecture provided by Django, ensuring clean code organization and easy maintenance.

### Key Features

* User Registration and Login
* Furniture Category and Subcategory Management
* Product Listing with Images and Details
* Product Search
* Shopping Cart
* Wishlist
* Buy Now and Checkout
* Order Management
* Customer Address Management
* Responsive User Interface
* Admin Dashboard for Product and Order Management

### Technologies Used

* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
* **Backend:** Python, Django
* **Database:** MySQL
* **Template Engine:** Django Templates
* **Version Control:** Git & GitHub

### Development Tools

* PyCharm (IDE)
* XAMPP (MySQL Server)
* Git
* GitHub
* Google Chrome
* Windows Operating System

### Project Objective

The objective of this project is to provide a modern and efficient online furniture shopping platform that enables customers to browse products easily while allowing administrators to manage inventory and orders through a secure and organized system.
